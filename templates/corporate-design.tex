\section{Einleitung}
\sectionbar

Dieses Dokument definiert die visuellen Gestaltungsrichtlinien der Marke \textbf{IlmTech}.
Es stellt sicher, dass alle Kommunikationsmittel — von der Website über Drucksachen bis zu
Präsentationen — ein einheitliches, professionelles Erscheinungsbild aufweisen.

\subsection{Über IlmTech}

IlmTech bietet intelligente Lösungen in den Bereichen \textbf{Smart Home \& Energie}
sowie \textbf{KI \& Automation}. Der Sitz ist in Ilmenau, Thüringen. Die Marke steht für:

\begin{itemize}[leftmargin=2em]
  \item Technologische Kompetenz und Innovation
  \item Lokale Nähe und persönliche Betreuung
  \item Datenschutz und Transparenz
  \item Zukunftssichere, offene Lösungen
\end{itemize}

\section{Logo \& Wortmarke}
\sectionbar

\subsection{Logo (Bildmarke)}

Das IlmTech-Logo ist eine stilisierte Gebäude-Silhouette, die Smart Home und Technologie
symbolisiert. Es wird als Strichzeichnung in der Primärfarbe \textcolor{ilm-blue!80!black}{\textbf{Accent Blue}} verwendet.

\begin{center}
  \fbox{\includegraphics[height=40mm]{logo.pdf}}
  
  \vspace{6pt}
  {\small Bildmarke — Standardgröße, freistehend}
\end{center}

\subsection{Wortmarke}

Die Wortmarke besteht aus zwei Teilen:

\begin{center}
  \begin{tikzpicture}
    \node[inner sep=12pt, fill=ilm-bg, rounded corners=6pt] {%
      {\fontsize{28}{34}\selectfont\bfseries
        \textcolor{ilm-blue}{Ilm}\textcolor{white}{Tech}%
      }%
    };
  \end{tikzpicture}
  
  \vspace{6pt}
  {\small Wortmarke auf dunklem Hintergrund}

\begin{itemize}[leftmargin=2em]
  \item \textbf{„Ilm"} — In der Primärfarbe Accent Blue (\texttt{\#00D4FF})
  \item \textbf{„Tech"} — In Weiß (auf dunklem Grund) oder Dunkel (auf hellem Grund)
  \item Schrift: \textbf{Inter Bold}, keine Abstände, kein Bindestrich
\end{itemize}

\subsection{Kombinierte Marke}

Die bevorzugte Darstellung kombiniert Bildmarke und Wortmarke horizontal:

\subsection{Schutzzone}

Um das Logo herum ist eine Schutzzone von mindestens \textbf{50\%} der Logohöhe einzuhalten.
Kein Text, Grafik oder Seitenkante darf in die Schutzzone ragen.

\subsection{Nicht erlaubt}

\begin{itemize}[leftmargin=2em]
  \item Logo drehen, verzerren oder spiegeln
  \item Farbverlauf auf das Logo anwenden
  \item Logo mit niedrigem Kontrast auf unruhigem Hintergrund platzieren
  \item Wortmarke in einer anderen Schrift oder Groß-/Kleinschreibung setzen
  \item Einzelteile der kombinierten Marke trennen und umpositionieren
\end{itemize}

\section{Anwendungsbeispiele}
\sectionbar

\subsection{Briefkopf}

Der Briefkopf verwendet die kombinierte Marke (Logo + Wortmarke) rechts oben, eine dünne
Akzentlinie in Accent Blue unter dem Header, und die Kontaktdaten im Footer.

\begin{itemize}[leftmargin=2em]
  \item Klasse: \texttt{ilmtech-letter.cls} (KOMA-Script scrlttr2)
  \item Akzentlinie: 0.6pt, Accent Blue
  \item Footer: Firma · Inhaber · Adresse | Seite X von Y
\end{itemize}

\subsection{Präsentation}

Beamer-Folien verwenden den dunklen Hintergrund (\texttt{\#0B1121}) mit heller Schrift.

\begin{itemize}[leftmargin=2em]
  \item Theme: \texttt{beamerthemeilmtech.sty}
  \item Format: 16:9 (\texttt{aspectratio=169})
  \item Titelfolie: Logo + Wortmarke, Accent Blue Top-Bar
  \item Abschnittstrennfolien: Nummerierung + Abschnittsname zentriert
  \item Footer: Firmenname links, Seitenzahl rechts
  \item Handout-Variante: 2 Folien pro A4-Seite
\end{itemize}

\subsection{Website}

\begin{itemize}[leftmargin=2em]
  \item Framework: Next.js mit Tailwind CSS
  \item Farbvariablen: CSS Custom Properties in \texttt{:root}
  \item Animationen: Framer Motion (subtile Einblendungen, keine ablenkenden Effekte)
  \item Responsiv: Mobile-first, Breakpoints bei \texttt{sm} (640px), \texttt{md} (768px), \texttt{lg} (1024px)
\end{itemize}
